# ニコタイチャット【YouTube|Twitch】

YouTube Live の視聴ページで拡張機能アイコンを押すと、チャット送信用パネルをページ内に表示する Chrome 拡張機能です。

## 開発

```sh
pnpm install
pnpm build
```

Chrome の `chrome://extensions` で「デベロッパー モード」を有効化し、「パッケージ化されていない拡張機能を読み込む」から `dist` を選択します。

## 使い方

1. ログイン済みの YouTube Live 視聴ページを開きます。
2. ページ内のチャット欄を表示します。
3. 拡張機能アイコンを押します。
4. ページ右下のパネルに入力し、「送信」または Ctrl/Cmd + Enter を押します。

パネルはタイトルバーのドラッグで移動できます。拡張機能アイコンの再クリックまたはパネル右上の閉じるボタンで非表示にできます。

拡張機能アイコンを右クリックして「オプション」を開くと、ページ内パネルと別ウィンドウの表示方式を切り替えられます。

この版は YouTube の非公開 API や認証トークンを直接使用せず、ページ内の通常のチャットUIを介して送信します。YouTube のDOM変更により動作しなくなる場合があります。
