# ニコタイチャット【YouTube|Twitch】プライバシーポリシー

最終更新日: 2026年7月21日

ニコタイチャット【YouTube|Twitch】（以下「本拡張機能」）は、YouTube LiveおよびTwitchのチャットへ、利用者が入力して送信を確定したメッセージを送るためのChrome拡張機能です。

## 取り扱う情報

本拡張機能は、機能提供に必要な範囲で以下の情報を取り扱います。

- 利用者が入力し、送信操作を行ったチャットメッセージ
- 開いているYouTubeまたはTwitchの配信ページURLとチャンネル識別情報
- Twitch OAuthアクセストークンおよび更新トークン
- 送信キー、文字サイズなどの拡張機能設定
- YouTubeへの送信時に、YouTubeページ内で一時的に参照するログインセッション情報

## 利用目的と送信先

これらの情報は、利用者が選択した配信のチャットへメッセージを送信し、設定と認証状態を維持する目的にのみ使用します。

- YouTube向けメッセージと必要な認証情報は、YouTubeのサーバーにのみ送信されます。
- Twitch向けメッセージと必要な認証情報は、Twitchの公式OAuthおよびAPIサーバーにのみ送信されます。
- 開発者が管理するサーバーへの送信は行いません。

## 保存

送信キー、文字サイズ、Twitch認証トークンは、利用者のブラウザー内のChrome Storageに保存されます。チャットメッセージの履歴は保存しません。

## 第三者提供

機能提供に必要なYouTubeおよびTwitchへの送信を除き、利用者情報を第三者へ販売、貸与、広告目的で提供することはありません。

## 削除

本拡張機能をChromeから削除すると、Chrome Storageに保存された設定と認証情報も削除されます。Twitch側の認可は、Twitchの「他の接続」設定から取り消すことができます。

## セキュリティ

外部サービスとの通信にはHTTPSを使用します。Client Secret、YouTubeのCookie、チャット履歴を拡張機能内へ永続保存しません。

## Limited Use

本拡張機能による利用者データの使用は、Chrome Web Store User Data Policy（Limited Use要件を含む）に準拠します。
